# ♜ Projeto Xadrez-OO

> **"O xadrez, como a música e o amor, tem o poder de fazer os homens felizes."** – Siegbert Tarrasch

## 🚀 Visão Geral
Este é um projeto de **desenvolvimento contínuo** focado em transformar a paixão pelo xadrez em excelência técnica na programação. O objetivo é construir um motor de jogo completo em Java, aplicando os conceitos mais modernos de **Análise e Desenvolvimento de Sistemas (ADS)**.

Este projeto representa a intersecção perfeita entre um **hobby pessoal** e a **carreira profissional**, utilizando o jogo de xadrez como um laboratório para simular desafios reais de engenharia de software.

## 🛠️ Tecnologias e Conceitos
- **Linguagem:** Java (JDK 17+)
- **IDE:** IntelliJ IDEA
- **Paradigma:** Orientação a Objetos (POO)
- **Arquitetura:** Camadas (Modelo, Lógica e Aplicação)

## 🏗️ Estrutura de Pacotes (Reverse Domain)
O projeto utiliza a convenção profissional de nomenclatura para garantir a unicidade e organização do código:
- `br.com.xadrez.modelo`: Classes que representam os dados (Peça, Posição, Tabuleiro).
- `br.com.xadrez.logica`: O "cérebro" do jogo e regras de negócio.
- `br.com.xadrez.app`: Ponto de entrada do sistema.

## 📈 Desenvolvimento Contínuo
Este repositório segue um modelo de **implementações contínuas**. Novas funcionalidades e refatorações são aplicadas à medida que novos conceitos de engenharia são absorvidos.
- [ ] Modelagem base (Posição e Peça)
- [ ] Sistema de Tabuleiro 8x8
- [ ] Lógica de capturas e Cemitério
- [ ] Regras de movimentos específicos
- [ ] Interface de terminal interativa

## ⚖️ Filosofia do Projeto
Construir este software não é apenas sobre o jogo em si, mas sobre a fundação de uma carreira. Cada classe criada é um passo para dominar ferramentas de mercado, garantindo que o aprendizado seja sólido, escalável e pronto para desafios profissionais.
